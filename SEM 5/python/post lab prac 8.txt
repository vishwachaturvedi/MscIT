QUE 1 : What is function?
ANS 1 : A function is a block of code which only runs when it is called.You can pass data, known as parameters, into a function.A function can return data as a result.


QUE 2 : What is Recursive function?
ANS 2 : Python also accepts function recursion, which means a defined function can call itself.Recursion is a common mathematical and programming concept. It means that a function calls itself. This has the benefit of meaning that you can loop through data to reach a result.
